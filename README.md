# YogaFire
코로나 이후 집에서 할 수 있는 홈트레이닝에 대한 관심도가 증가했고, 바쁜 현대인들이 자신의 시간에 맞게 할 수 있는 홈트레이닝에 대한 관심도가 증가했다고 판단.
홈트레이닝과 관련된 서비스를 제작하기로 목표를 잡고 어떤 운동을 선택할까 고민하다가 집에서 간단하게 따라할 수 있는 요가영상 관련 사이트를 만들어보자는 의견으로 '요가 관련 사이트'를 제작.
